## Задание 9 ЕГЭ: Чередующиеся гласные в корнях слов

**Важное правило:** Гласные в этих корнях **не проверяются ударением**. Их написание зависит от строго определённых условий.

---

### Типы чередований и правила

#### 1. Чередование **-е- / -и-**

| Корень | Правило | Примеры |
|--------|---------|---------|
| **-бер- / -бир-** | В корне пишется **-и-**, если после корня стоит суффикс **-а-**. В остальных случаях пишется **-е-**. | **забира́ть**, **соберу́**, **поберу́шка** |
| **-дер- / -дир-** | | **выдира́ть**, **уде́рет** |
| **-мер- / -мир-** | | **умира́ть**, **за́мер** |
| **-пер- / -пир-** | | **запира́ть**, **запере́ть** |
| **-тер- / -тир-** | | **подтере́ть**, **затира́ть** |
| **-блест- / -блист-** | | **блесте́ть**, **блиста́ть** |
| **-стел- / -стил-** | | **застила́ть**, **стели́ть** |
| **-чет- / -чит-** | | **вычита́ть**, **учитыва́ть**, **почита́ть** (старших) <br> **Исключения:** *сочета́ние, че́та, бракосочета́ние* |
| **-жег- / -жиг-** | | **заже́чь**, **выжига́ть** |

---

#### 2. Чередование **-о- / -а-**

##### **а) Зависит от последующей буквы**

| Корень | Правило | Примеры | Исключения |
|--------|---------|---------|------------|
| **-лаг- / -лож-** | Перед **-г-** пишется **-а-**, перед **-ж-** пишется **-о-**. | **предлага́ть** — **предложи́ть** | **поло́г** |
| **-кас- / -кос-** | Перед **-с-** пишется **-о-**, если после корня стоит суффикс **-а-** — пишется **-а-**. | **каса́ться** — **прикоснове́ние** | — |
| **-раст- / -ращ- / -рос-** | Перед **-ст-** и **-щ-** пишется **-а-**, в остальных случаях пишется **-о-**. | **вы́растить**, **выра́щивать**, **вы́росли** | **ро́сток, Росто́в, Ростисла́в, о́трасль** |
| **-скак- / -скоч-** | Перед **-к-** пишется **-а-**, перед **-ч-** пишется **-о-**. | **скака́ть**, **подскочи́ть**, **скачу́** | — |

##### **б) Зависит от ударения**

| Корень | Правило | Примеры | Исключения |
|--------|---------|---------|------------|
| **-гар- / -гор-** | Под ударением пишется **-а-**, без ударения — **-о-**. | **загоре́лый**, **подгоре́ть**, **зага́р** | **при́гарь, вы́гарки, и́згарь** |
| **-зар- / -зор-** | Под ударением пишется то, что слышится, без ударения — **-а-**. | **заря́**, **зарни́ца**, **зо́рька** | **зорева́ть** |
| **-клан- / -клон-** | Под ударением пишется то, что слышится, без ударения — **-о-**. | **склоне́ние**, **раскла́нялся** | — |
| **-твар- / -твор-** | Под ударением пишется то, что слышится, без ударения — **-о-**. | **благотвори́тельный**, **сотворе́ние** | **у́тварь** |

##### **в) Зависит от значения**

| Корень | Правило | Примеры | Исключения |
|--------|---------|---------|------------|
| **-мак- / -мок-** | **-мак-** — «погружать в жидкость»<br>**-мок-** — «пропускать жидкость» | **мака́ть** сухарь, **промо́кнуть** под дождём | — |
| **-равн- / -ровн-** | **-равн-** — «одинаковый, равный»<br>**-ровн-** — «гладкий, ровный, прямой» | **уравня́ть** в правах, **вы́ровнять** дорогу | **равни́на, поравня́ться, ро́весник, поро́вну, равне́ние** |

---

### Особые случаи для запоминания

| Корень | Правило | Примеры | Исключения |
|--------|---------|---------|------------|
| **-плав- / -плов- / -плыв-** | Почти всегда пишется **-а-**. Буква **-о-** пишется **только** в словах **плове́ц, пловчи́ха**. Буква **-ы-** пишется только в слове **плывуны́**. | **пла́вать**, **плыву́**, **жук-плавуне́ц** | **плове́ц, пловчи́ха, плывуны́** |
| **-им- / -ин-**, **-а- / -я-** | В глаголах, если после корня стоит суффикс **-а-**, то в корне пишется **-им-/-ин-**. | **сжима́ть**, **занима́ть**, **поднима́ть**, **понима́ть**, **начина́ть** | — |
