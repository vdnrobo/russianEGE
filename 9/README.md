### Задание 9. Орфография: чередующиеся гласные в корнях слов

**Важное правило:** Гласные в этих корнях **не проверяются ударением**. Их написание зависит от строго определённых условий.

---

### 1. Чередование **-е- / -и-**

| Корень | Правило | Примеры |
| :--- | :--- | :--- |
| **-бер- / -бир-** | В корне пишется **-и-**, если после корня стоит суффикс **-а-**. В остальных случаях пишется **-е-**. | **брать** — **бирать**<br>**разберу** — **разбирать** |
| **-дер- / -дир-** | | **придёртся** — **придираться** |
| **-мер- / -мир-** | | **замерло** — **замирать** |
| **-пер- / -пир-** | | **отпереть** — **отпирать** |
| **-тер- / -тир-** | | **протереть** — **протирать** |
| **-блест- / -блист-** | | **блеснуть** — **блистать** |
| **-стел- / -стил-** | | **расстелить** — **расстилать** |
| **-чет- / -чит-** | | **вычет** — **вычитать** |

---

### 2. Чередование **-о- / -а-**

#### **а) Зависит от последующей буквы (согласной или суффикса)**

| Корень | Правило | Примеры | Исключения |
| :--- | :--- | :--- | :--- |
| **-лаг- / -лож-** | Перед **-г-** пишется **-а-**, перед **-ж-** пишется **-о-**. | **предлагать** — **предложить** | **полог** |
| **-кос- / -кас-** | Перед **-с-** пишется **-о-**, если после корня стоит суффикс **-а-** — пишется **-а-**. | **прикасаться** — **коснуться** | — |
| **-рос- / -раст- / -ращ-** | Перед **-ст-** и **-щ-** пишется **-а-**, в остальных случаях пишется **-о-**. | **прирасти**, **переросла**, **выращенный** | **росток, Ростов, Ростислав, ростовщик, отрасль** |
| **-скак- / -скоч-** | Перед **-к-** пишется **-а-**, перед **-ч-** пишется **-о-**. | **заскакивать**, **заскочить**, **прискакать** | **скачок, скачу** |

#### **б) Зависит от ударения**

| Корень | Правило | Примеры | Исключения |
| :--- | :--- | :--- | :--- |
| **-гор- / -гар-** | Под ударением пишется **-а-**, без ударения — **-о-**. | **нагореть** — **нагар**, **загар** | **пригар, изгар** |
| **-зор- / -зар-** | Под ударением пишется то, что слышится (и **-о-**, и **-а-**), без ударения — **-а-**. | **зарево** — **зори**, **озарять** — **заря** | **зоревать** |
| **-клон- / -клан-** | Под ударением пишется то, что слышится, без ударения — **-о-**. | **поклонение**, **преклонный**, **наклонить** | — |
| **-твор- / -твар-** | Под ударением пишется то, что слышится, без ударения — **-о-**. | **творить**, **творение**, **благотворный** | **утварь** |

#### **в) Зависит от значения**

| Корень | Правило | Примеры | Исключения |
| :--- | :--- | :--- | :--- |
| **-мак- / -мок-** | **-мак-** — значение «погружать в жидкость».<br>**-мок-** — значение «пропускать жидкость, становиться мокрым». | **макать** сухарь, **промокнуть** под дождём | — |
| **-ровн- / -равн-** | **-ровн-** — значение «гладкий, ровный, прямой».<br>**-равн-** — значение «одинаковый, равный». | **выравнивать** дорогу, **уравнять** в правах, **равнобедренный** | **равнина, уровень, поровну** |

### 3. Особые случаи для запоминания

| Корень | Правило | Примеры | Исключения |
| :--- | :--- | :--- | :--- |
| **-плав- / -плов-** | Почти всегда пишется **-а-**. Буква **-о-** пишется **только** в словах **пловец, пловчиха**. Буква **-ы-** пишется только в слове **плывун**. | **плавание, поплавок, плавучий** | **пловец, пловчиха, плывун** |
| **-жа- / -жим-** <br> **-ня- / -ним-** | В глаголах, если после корня стоит суффикс **-а-**, то в корне пишется **-им-**/**-ин-**. | **сжать** — **сжимать**<br>**начать** — **начинать**<br>**занять** — **занимать** | — |
