# Задание 5. Паронимы

## Существительные

| Паронимы | Ключевое отличие в значении | Пример употребления |
| :--- | :--- | :--- |
| **Абонемент** – **Абонент** | Документ на право пользования / Владелец документа | Купить **абонемент** в бассейн / **Абонент** телефонной сети |
| **Адресат** – **Адресант** | Тот, кому адресовано / Тот, кто адресует | Письмо дошло до **адресата** / Установить личность **адресанта** |
| **Диктант** – **Диктат** | Вид письменной работы / Жёсткое навязывание | Написать контрольный **диктант** / Политический **диктат** |
| **Невежа** – **Невежда** | Грубый, невоспитанный человек / Необразованный, несведущий человек | Грубить — значит быть **невежей** / Он полный **невежда** в физике |

## Прилагательные

| Паронимы | Ключевое отличие в значении | Пример употребления |
| :--- | :--- | :--- |
| **Болотный** – **Болотистый** | Относящийся к болоту / Изобилующий болотами | **Болотная** трава / **Болотистая** местность |
| **Великий** – **Величественный** | Очень большой, выдающийся / Торжественный, полный достоинства | **Великий** поэт / **Величественный** вид гор |
| **Действенный** – **Действительный** | Эффективный / Реально существующий | **Действенные** меры / **Действительная** причина |
| **Дружеский** – **Дружественный** | Относящийся к другу / Основанный на дружбе (часто о государствах) | **Дружеский** совет / **Дружественные** страны |
| **Экономичный** – **Экономический** | Позволяющий экономить / Относящийся к экономике | **Экономичный** автомобиль / **Экономический** кризис |

## Глаголы

| Паронимы | Ключевое отличие в значении | Пример употребления |
| :--- | :--- | :--- |
| **Одеть** – **Надеть** | Кого-то (одеть ребёнка) / Что-то на себя (надеть пальто) | **Одеть** малыша на прогулку / **Надеть** платье на праздник |
| **Освоить** – **Усвоить** | Научиться пользоваться, включить в деятельность / Понять, запомнить | **Освоить** новую программу / **Усвоить** правило |
| **Выплатить** – **Оплатить** – **Заплатить** | Выдать плату (деньги) / Внести плату (за что-то) / Отдать плату (за что-то или *кому-то*) | **Выплатить** зарплату / **Оплатить** счёт / **Заплатить** в кассу |
