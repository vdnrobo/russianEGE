## Задание 16. Знаки препинания в сложном предложении (СП)

### 1. Однородные члены предложения `[-=], и [-=]`

*Мама читает роман, и папа жарит котлетки*

#### Правило:
Запятая **НЕ СТАВИТСЯ** между однородными членами, если есть **общие условия**:

| Условие | Пример |
|---------|---------|
| **Общий второстепенный член (ОВЧ)** | **На кухне** мама читает роман и папа жарит котлетки |
| **Общий вводный элемент** | **К счастью**, мама читает роман и папа жарит котлетки |
| **Вопрос** | Кто читал роман и кто жарил котлетки? |
| **Восклицание** | Мама жарит котлетки и папа читает роман! |

### 2. Особые случаи союзов

#### Типы союзов и правила:

| Тип союзов | Правило | Пример |
|------------|---------|---------|
| **Повторяющиеся** | Запятая ставится между всеми однородными членами | Я люблю **и** яблоки, **и** груши, **и** мандарины |
| **Двойные** | Запятая ставится перед второй частью союза | **Как** яблоки, **так и** груши |
| **Разные ряды однородных** | Запятая между разными рядами | Я люблю есть яблоки и груши, и читать романы |

### 3. Однородные и неоднородные определения (ОЧП)

#### Различия:

| Тип определений | Признаки | Пример |
|-----------------|----------|---------|
| **Однородные** | • Эпитеты<br>• Произносятся с перечислительной интонацией<br>• Обозначают схожие признаки | **Тёмные, мрачные** тучи |
| **Однородные** | • Образуют причинно-следственную связь | В лицо била **острая, мелкая** пыль (мелкая, потому что острая) |
| **Однородные** | • Прилагательное + причастный оборот | **Чёрные, гладко зачёсанные назад** волосы |
| **Неоднородные** | • Обозначают разные признаки<br>• Произносятся без перечислительной интонации | **Большой кирпичный** дом |
